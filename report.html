Story Being Told by my website

My website is a home for everyone who is obseesed with new forms of technology (particulary gadgets). It provides everything a technology enthusuaist would enjoy and engage in from lots of interesting stories on the new where some form of technology has made an appearance to an expo which is aimed directly at tech lovers. The structure of my website is very simple and each page consists of section tags segregating all of the different parts of it. Further into the structure I have used DIV tags to divide all of the different elements on each page.

3 things that inpired me whilst creating this webiste were:

1.The guest speaker who spoke to us about web development in one of the lectures who also marked my webpage for the group website project

2.A selection of web developers who are professionals and make a series of youtube videos aimed at beginners like myself on how to go about creating a good website

3.I was also inpsired by websites such as www.newscientist.com as I went to these to get the content for my articles and observed how they had layed out their content.

3 things that make my website accessable

1.I have added alt tags to all of my images so the blind can still see a description of them.

2. The elemnts in my website are of a good size and should be visible to all different types of people with different eye sight levels

3.My website doesnt contain any flashing images as this may effect people who suffer from epileptic seizures which is a seriouse health issue.

3 things that make my website usuable

1. one way in which my website is usuable is that all of the content is places neatly into differnt places making it easy for people to find what threy are looking for

2.the website contains simple javascript coding and doesnt contain anything that would not be compatible on older web browsers on older operating systems

3.my websites responsiveness partially works and people who are using tablets and for some pages mobile phones aswell, can see all of the content on the web pages.

3 things i learnt when creating this website

1. I had to leanr how to create an image carousel using basic javascript code. I did this by making a search onn youtube and watching a video on a web developers channel

2.I had to figure out how to make images and buttons links to other pages using a basic javascript line of code. I found this out from a contributer to stack overflow.

3. I had to learn how to use jquery to show and hide div tags as i needed this core peiece of functionality for the news articles. I learnt this from a person on youtube too.
